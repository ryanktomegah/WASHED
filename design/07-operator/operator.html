Operator console — desktop

Hi-fi des écrans clés côté opérateur. Outil dense, raccourcis clavier prioritaires (palette de commandes ⌘K), vue à 3 colonnes (sidebar — table — drawer). Le jugement humain au cœur : aucun matching automatisé non-supervisé.

Vocabulaire visuel — AdminSurface neutre #FAFAFD. Violet #3B1F7A comme accent unique (action principale, lien actif, mise en évidence). Densité Linear-grade : 12px texte, lignes de tableau 36-40px de hauteur. Tabular-nums sur tous les chiffres. Aucune carte arrondie inutile dans les vues opérationnelles — on optimise pour le scanning rapide. Ombres uniquement pour overlays modaux.
